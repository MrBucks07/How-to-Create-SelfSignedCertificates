/**
 * How to create Self Signed Certificate step by step.
 * Author : Kartik Panchal
 */

Step 1
Download Openssl which we use to create certificate.
link : https://www.openssl.org/
Command : openssl version or openssl version -a   --> Shows version of openssl and other information. 

Step 2
Create key pair
Command : openssl genrsa -out (keypairfilename.key) 2048

Step 3
Extract public key from key pair
Command : openssl rsa -in (keypairfilename.key) -pubout -out (publickeyfilename.key)

Step 4
Create certificate signing request for key pair.
Command : openssl req -new -key (keypairfilename.key) -out (csrfilename.csr)

Step 5
Checking that inforamation given during csr request is correct.
Command : openssl req -text -in (csrfilename.csr) -noout -verify

Step 6
Signing Certificate (Self)
Command : openssl x509 -in (csrfilename.csr) -out (crtfilename.crt) -req -signkey (keypairfilename.key) -days 365
